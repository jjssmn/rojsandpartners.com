DNS Analyse: Google DKIM Record

Wat betekent dit?

Dit is een DKIM (DomainKeys Identified Mail) record voor Google Workspace.

Het zorgt ervoor dat je e-mails:

  • ✅ Niet in de spam belanden (verifieert dat jij de echte afzender bent)
  • ✅ Veilig zijn (voorkomt spoofing/vervalsing)
  • ✅ Betrouwbaar zijn voor Gmail, Outlook, etc.

⚠️ Belangrijk: niet verwijderen!

Critical e‑mail beveiliging

Als je dit verwijdert:

  • Je e-mails (van info@rojsandpartners.com) gaan sneller naar spam‑folders.
  • Google Workspace mail werkt minder goed.
  • Je domeinreputatie daalt merkbaar.

Technische details

Veld Waarde Uitleg
Host google._domainkey Specifiek subdomein voor Google Workspace mail‑services.
Type TXT Tekstrecord dat gebruikt wordt voor e‑mailverificatie.
Data v=DKIM1; k=rsa; p=... De publieke sleutel waarmee ontvangende mailservers je DKIM‑handtekening controleren.

Conclusie

Laat dit record staan! Het is essentieel voor je Google Workspace e‑mailfunctionaliteit en je domeinreputatie.

Dit DKIM‑record heeft geen invloed op je website‑hosting via GitHub Pages, dus je kunt je website‑DNS (CNAME/ALIAS) er gewoon naast instellen zonder problemen.

Tip: wijzig alleen de waarde p=... als je in Google Admin Console een nieuwe DKIM‑sleutel genereert; de hostnaam google._domainkey blijft dan gelijk.
